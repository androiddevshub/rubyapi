# Build API using ruby on rails.

# Setup

1) Clone the repository
2) run `bundle update`
2) run `rails db:migrate`
3) run `rails s`

Routes-
1) users - to view all users
2) user/register - to register user
3) user/send_otp - to send otp
4) user/login - to login user
